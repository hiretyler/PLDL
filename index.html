Setup Guide
1

Download the server.js file from Claude, then install dependencies:

npm install express cors
2

Install yt-dlp (the YouTube downloader engine):

brew install yt-dlp   or   pip install yt-dlp
3

Install ffmpeg (for merging video + audio streams):

brew install ffmpeg   or visit ffmpeg.org
4

Start the backend server:

node server.js
5

Paste a YouTube playlist URL above and click Analyze.
